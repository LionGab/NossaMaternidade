# ⚡ PLANO DE AÇÃO RÁPIDO - Versão Resumida

**Para quem quer começar AGORA sem ler tudo.**

---

## 🎯 3 SEMANAS PARA PUBLICAR

### SEMANA 1: Setup + Backend

**Dia 1-2:**
```bash
npm install
cp .env.example .env
# Editar .env com credenciais
npm run type-check
```

**Dia 3-4:**
- Configurar Supabase
- Aplicar schema
- Configurar RLS

**Dia 5:**
- Implementar guardrails IA
- Testar tudo

---

### SEMANA 2: Build + Assets

**Dia 1-2:**
- Configurar EAS
- Criar contas Apple/Google Developer

**Dia 3-4:**
- Criar screenshots
- Development build
- Testar em devices

**Dia 5:**
- Production build
- Testes finais

---

### SEMANA 3: Publicação

**Dia 1:**
- Preparar metadata
- Upload screenshots

**Dia 2:**
- Submeter para lojas

**Dia 3-7:**
- Aguardar aprovação
- Responder rejeições (se houver)

---

## 🚀 COMANDOS ESSENCIAIS

```bash
# Setup
npm install

# Validação
npm run validate:production
```
