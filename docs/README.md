## 📝 Scripts Úteis

### Validação
```bash
# Validar prontidão para produção
npm run validate:production

# Validar design tokens
npm run validate:design

# Validar TypeScript
npm run type-check

# Health check completo
npm run health-check
```

### Build e Deploy
```bash
# Development build
npm run build:dev

# Production build
npm run build:production

# Submeter para lojas
npm run submit:ios
npm run submit:android
```

### Migração
```bash
# Migrar cores hardcoded
node scripts/migrate-hardcoded-colors.js --all
```
