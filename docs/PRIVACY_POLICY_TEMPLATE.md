## 1. Introdução

Bem-vinda ao **Nossa Maternidade** ("nós", "nosso" ou "app"). Respeitamos sua privacidade e estamos comprometidos em proteger seus dados pessoais. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações quando você usa nosso aplicativo móvel.

**⚠️ IMPORTANTE - AVISO MÉDICO:**
Este aplicativo fornece informações educacionais e suporte emocional relacionados à maternidade, mas **NÃO substitui consulta médica profissional**. Em caso de emergência médica, procure imediatamente um profissional de saúde ou ligue para 192 (SAMU).

Ao usar o Nossa Maternidade, você concorda com a coleta e uso de informações de acordo com esta política.

## 5. Compartilhamento de Dados

### 5.1 NÃO Vendemos Seus Dados

**Nunca vendemos, alugamos ou comercializamos seus dados pessoais com terceiros.**

### 5.2 Processadores de Dados (Terceiros Necessários)

Compartilhamos dados apenas com provedores de serviços essenciais:

| Provedor | Finalidade | Dados Compartilhados | Localização |
|----------|------------|---------------------|-------------|
| **Supabase** | Hospedagem de banco de dados | Todos os dados do app | EUA/EU |
| **Google (Gemini AI)** | Chat com IA | Mensagens do chat (não armazenadas pelo Google) | EUA |
| **OneSignal** | Notificações push | ID do dispositivo, token push | EUA |
| **Sentry** | Monitoramento de erros | Logs de erro (anônimos), info do dispositivo | EUA |
| **Expo/EAS** | Build e atualizações | Dados técnicos do app | EUA |

**Importante:** Estes provedores são **processadores de dados**, não controladores. Eles não podem usar seus dados para outros fins.

### 5.3 Outros Compartilhamentos

Podemos compartilhar dados em casos de:
- **Requisição Legal:** Ordem judicial, investigação policial
- **Proteção de Direitos:** Defender nossos direitos legais
- **Venda da Empresa:** Transferência de ativos (você será notificada)
- **Consentimento Explícito:** Quando você autorizar

## 6. Segurança dos Dados

### 6.1 Medidas de Segurança

Implementamos proteções técnicas e organizacionais:

**Criptografia:**
- ✅ HTTPS/TLS 1.3 para todos os dados em trânsito
- ✅ Criptografia de banco de dados em repouso (AES-256)
- ✅ Senhas armazenadas com hash bcrypt

**Controles de Acesso:**
- ✅ Autenticação de dois fatores (para admin)
- ✅ Row Level Security (RLS) no banco de dados
- ✅ Acesso baseado em permissões

**Monitoramento:**
- ✅ Logs de auditoria
- ✅ Detecção de tentativas de invasão
- ✅ Backups regulares

### 6.2 Limitações

Nenhum sistema é 100% seguro. Fazemos o melhor para proteger seus dados, mas não podemos garantir segurança absoluta. Use uma senha forte e única.

### 7.4 Restrição de Processamento
- ✅ Solicitar limitação de como usamos seus dados
- ✅ Email: privacy@nossamaternidade.com.br

### 7.5 Oposição

### 7.6 Retirar Consentimento

## 8. Retenção de Dados

### 8.1 Enquanto Sua Conta Estiver Ativa
- Dados de perfil, posts, hábitos, chat: mantidos indefinidamente

### 8.2 Após Exclusão da Conta
- Dados deletados permanentemente: **30 dias**
- Backups mantidos por motivos técnicos: **90 dias** (então deletados)

### 8.3 Dados Específicos
| Tipo de Dado | Período de Retenção |
|--------------|---------------------|
| Mensagens do chat | Enquanto conta ativa |
| Posts da comunidade | Enquanto conta ativa (ou até você deletar) |
| Logs de erro | 90 dias |
| Dados de analytics | 24 meses (agregados, não identificáveis) |
| Dados fiscais | 5 anos (obrigação legal brasileira) |

- Parar de usar funcionalidades de saúde
