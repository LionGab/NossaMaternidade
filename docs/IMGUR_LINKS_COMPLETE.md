## Notas

- Todos os links são URLs diretas de imagens no formato `https://i.imgur.com/{ID}.{ext}`
- As imagens são usadas principalmente como assets estáticos do app
- Muitas imagens são reutilizadas entre diferentes projetos

## Ação Recomendada

Para o projeto atual, considere:
1. Migrar assets estáticos para o sistema de upload do Imgur
2. Ou manter as imagens em `assets/` locais
3. Usar as URLs do Imgur apenas para uploads dinâmicos de usuários
