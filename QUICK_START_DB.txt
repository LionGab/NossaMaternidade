📋 PASSO A PASSO RÁPIDO (2 minutos)

1️⃣  Abra o SQL Editor do Supabase:
   👉 https://supabase.com/dashboard/project/mnszbkeuerjcevjvdqme/sql

2️⃣  Aplique o Schema:
   ▪ Clique em "New query"
   ▪ Abra o arquivo: supabase/schema.sql
   ▪ Copie TODO o conteúdo (Ctrl+A, Ctrl+C)
   ▪ Cole no SQL Editor (Ctrl+V)
   ▪ Execute (Ctrl+Enter ou botão "Run")
   ▪ Aguarde ~5 segundos

3️⃣  Aplique os Dados Iniciais:
   ▪ Crie uma nova query (botão "+")
   ▪ Abra o arquivo: supabase/seed.sql
   ▪ Copie TODO o conteúdo
   ▪ Cole e Execute

4️⃣  Verifique se funcionou:
   ▪ Acesse: Table Editor
   ▪ Deve ver 13 tabelas
   ▪ Clique em "habits" → deve ter 10 registros

5️⃣  Teste o app:
   npm start
