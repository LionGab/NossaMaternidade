# ⚡ Quick Start - Supabase em 5 Minutos

## 🎯 O Que Você Precisa Fazer AGORA

### 0️⃣ Instalar Dependências (30 segundos)

```bash
npm install
```

**✅ Dependência instalada:** `base64-arraybuffer` (para upload de avatares)

---

### 1️⃣ Aplicar Schema (2 minutos)

```bash
# 1. Acesse o Supabase Dashboard
open https://app.supabase.com

# 2. Vá em SQL Editor → New Query

# 3. Copie e cole TODO o conteúdo de:
supabase/schema.sql

# 4. Clique em RUN
```

---

### 2️⃣ Popular Dados (1 minuto)

```bash
# No SQL Editor → New Query

# Copie e cole TODO o conteúdo de:
supabase/seed.sql

# Clique em RUN
```

---

### 3️⃣ Testar Conexão (30 segundos)

```bash
npx ts-node scripts/test-supabase-connection.ts
```

**Resultado esperado:**
```
✅ Conexão estabelecida! Encontrados 10 hábitos.
✅ Encontrados 24 marcos de desenvolvimento.
🎉 Supabase está configurado corretamente!
```

---

### 4️⃣ Iniciar App (30 segundos)

```bash
npm start
```

---

## ✅ Pronto!

Se os testes passaram, seu backend Supabase está funcionando perfeitamente!
