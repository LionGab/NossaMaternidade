## 🚀 Comandos Rápidos

### Verificar Prontidão
```bash
npm run check-ready
```

### Build de Produção
```bash
# iOS
npm run build:ios

# Android
npm run build:android

# Ambos
npm run build:production
```

### Submissão
```bash
# iOS
npm run submit:ios

# Android
npm run submit:android
```

## ✅ Checklist Essencial

### Antes de Começar
- [ ] Apple Developer Account ($99/ano)
- [ ] Google Play Console ($25 única)
- [ ] EAS login configurado

### Assets Necessários
- [ ] Ícone 1024x1024px
- [ ] Splash screen
- [ ] Screenshots iOS (3-5)
- [ ] Screenshots Android (2-8)
- [ ] Feature graphic Android (1024x500px)

### Documentação Legal
- [ ] Privacy Policy publicada (URL)
- [ ] Terms of Service publicados (URL)
- [ ] Data Safety form (Google Play)

### Builds
- [ ] Build preview testado
- [ ] Build produção gerado
- [ ] Testado em dispositivos reais
